# Версии и перезапуск

`tunnelctl` использует три независимые версии:

- версия приложения (`tunnelctl version` и `doctor`);
- версия локального управляющего протокола и state-файла;
- версия схемы `tunnelctl.json`.

Версия приложения записывается в журнал при запуске и в безопасный state-файл. Команда `status` показывает версию работающего экземпляра, а не только версию вызванного CLI.

Конфигурация сохраняется с полем `version`. Файл без этого поля считается legacy-схемой 0, последовательно мигрируется в памяти и при следующем сохранении записывается в актуальной схеме. Файл с более новой неизвестной схемой не открывается, чтобы не потерять данные.

## Перезапуск после замены бинарника

Автоматического скачивания и установки обновлений нет. Новый бинарник устанавливается пользователем или пакетным менеджером, после чего выполняется:

```text
tunnelctl restart
```

Команду нужно запускать именно тем бинарником, который должен стать рабочим. Он передаёт свой абсолютный путь старому управляемому экземпляру через локальный канал. Старый экземпляр:

1. корректно завершает принадлежащий ему дочерний `ssh`;
2. ждёт освобождения SOCKS-порта;
3. закрывает управляющий канал и удаляет старый state-файл;
4. запускает новый бинарник с прежней целью, режимом `--watch`/`--no-watch` и абсолютным путём конфигурации.

На Linux и Termux используется замена текущего процесса, поэтому сохраняется PID и владение со стороны `systemd`. На Windows запускается новый процесс с попыткой выйти из Job Object Планировщика заданий. Вызывающая команда ждёт новый state и успешный health-check через SOCKS; отсутствие подтверждения считается ошибкой.

## Метки времени

Строки журнала и операционный консольный вывод используют локальное время в формате:

```text
YYYY-MM-DD HH:MM:SS.mmm [УРОВЕНЬ] сообщение
```

Интерактивные запросы, справка, вывод версии и содержимое, создаваемое `autostart print`/`--dry-run`, не префиксуются, чтобы не ломать ввод и копирование файлов. Вывод самого `ssh`, включая запросы пароля и проверку ключа хоста, остаётся прямым и не перехватывается.
