# План развития tunnelctl

## MVP 0.1

- Стартовые скрипты `start.sh`, `start.cmd`, `start.ps1`.
- Проверка и установка Go с явным выводом команды.
- Конфиг в стандартном месте ОС.
- Импорт `ssh -D` команд из истории.
- Генерация алиасов.
- Управляемый запуск системного `ssh` без `-f`.
- Health-check SOCKS и перезапуск.
- Ярлыки Windows/Linux/Termux.

## Следующие задачи

1. Встроенный SSH backend на Go без внешнего `ssh`.
2. Более аккуратная диагностика владельца занятого порта.
3. GitHub Actions сборки под Linux/Windows/Android Termux.
4. Улучшенный bootstrap с ручным добавлением профиля.
5. Release-архивы с готовыми бинарниками.
